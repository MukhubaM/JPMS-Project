# JPMS Project

##  Overview
This project demonstrates how the **Java Platform Module System (JPMS)** works by defining modules with `module-info.java`, managing dependencies and enforcing strong encapsulation between packages.

---

##  Features
- Showcases modular design in Java.
- Demonstrates `requires`, `exports` and `opens` keywords.
- Highlights dependency management between modules.
- Enforces encapsulation and clear boundaries.

---

##  Tech Stack
- **Java (JDK 17+)**
- **JPMS (Java Platform Module System)**
- **Maven** for build management
- **IntelliJ IDEA** for development

---

##  Project Structure
- `module-info.java` → Defines module dependencies and exports.
- `core/` → Core functionality module.
- `service/` → Service layer module.
- `app/` → Main application module demonstrating usage.
